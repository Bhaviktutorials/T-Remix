 ### Installation T-Remix

* `Commands` for termux
```
$ termux-setup-storage

$ pkg update && pkg upgrade && pkg install git -y

$ git clone https://github.com/Bhaviktutorials/T-Remix

$ ls

$ cd T-Remix

$ ls

$ chmod +x *

$ bash t-remix.sh
```
